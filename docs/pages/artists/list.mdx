# List Artists

Retrieve artists accessible to a user. Supports filtering by organization or showing only personal (non-organization) artists.

## Endpoint

```http
GET https://api.recoupable.com/api/artists
```

## Parameters

| Name      | Type   | Required | Description                                           |
| --------- | ------ | -------- | ----------------------------------------------------- |
| accountId | string | Yes      | The user's account ID                                 |
| orgId     | string | No       | Filter to artists in a specific organization          |
| personal  | string | No       | Set to `"true"` to show only personal (non-org) artists |

## Filtering Behavior

| Parameters                  | Returns                                    |
| --------------------------- | ------------------------------------------ |
| `accountId` only            | All artists (personal + all organizations) |
| `accountId` + `orgId=xyz`   | Only artists in that specific organization |
| `accountId` + `personal=true` | Only personal artists (not in any org)   |

## Request Examples

:::code-group

```bash [cURL]
# Get all artists
curl -X GET "https://api.recoupable.com/api/artists?accountId=user-uuid" \
  -H "Content-Type: application/json"

# Get artists in a specific organization
curl -X GET "https://api.recoupable.com/api/artists?accountId=user-uuid&orgId=org-uuid" \
  -H "Content-Type: application/json"

# Get only personal artists (not in any org)
curl -X GET "https://api.recoupable.com/api/artists?accountId=user-uuid&personal=true" \
  -H "Content-Type: application/json"
```

```javascript [JavaScript]
async function getArtists(accountId, options = {}) {
  const params = new URLSearchParams({ accountId });
  
  if (options.orgId) {
    params.append("orgId", options.orgId);
  }
  if (options.personal) {
    params.append("personal", "true");
  }

  const response = await fetch(
    `https://api.recoupable.com/api/artists?${params}`,
    {
      headers: {
        "Content-Type": "application/json",
      },
    }
  );

  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }

  return response.json();
}

// Usage examples
const allArtists = await getArtists("user-uuid");
const orgArtists = await getArtists("user-uuid", { orgId: "org-uuid" });
const personalArtists = await getArtists("user-uuid", { personal: true });
```

interface Artist {
  account_id: string;

interface ArtistsResponse {
  status: "success" | "error";
  artists: Artist[];
  message?: string;

  personal?: boolean;

async function getArtists(
  accountId: string,
  options: GetArtistsOptions = {}

  const params = new URLSearchParams({ accountId });
  
  if (options.orgId) {
    params.append("orgId", options.orgId);
  }
  if (options.personal) {
    params.append("personal", "true");
  }

  const response = await fetch(
    `https://api.recoupable.com/api/artists?${params}`,
    {
      headers: {
        "Content-Type": "application/json",
      },
    }
  );

  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }

  return response.json();
}

## Response Properties

| Property              | Type    | Description                                       |
| --------------------- | ------- | ------------------------------------------------- |
| status                | string  | Status of the request ("success" or "error")      |
| artists               | array   | List of artist objects                            |
| artists[].account_id  | string  | UUID of the artist account                        |
| artists[].name        | string  | Artist display name                               |
| artists[].image       | string \| null | Artist profile image URL                   |
| artists[].pinned      | boolean | Whether the user has pinned this artist           |
| artists[].socials     | array   | Social media profiles linked to the artist        |
| message               | string  | Error message (only present if status is "error") |

## Use Cases

- **Artist Sidebar**: Show list of accessible artists in navigation
- **Organization View**: Filter to show only artists managed by a specific org
- **Personal View**: Show only artists not shared with any organization
