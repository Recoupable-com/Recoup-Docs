# Delete Task

Delete an existing scheduled task by its `id`. The response returns only the status of the delete operation.

## Endpoint

```http
DELETE https://api.recoupable.com/api/tasks
```

## DELETE Parameters

| Name | Type   | Required | Description               |
| ---- | ------ | -------- | ------------------------- |
| id   | string | Yes      | UUID of the task to delete |

## Request Examples

:::code-group

```bash [cURL]
curl -X DELETE "https://api.recoupable.com/api/tasks" \
  -H "Content-Type: application/json" \
  -d '{
    "id": "aade2bce-55c7-468e-a606-c4e76fb2ea2a"
  }'
```

```python [Python]
import requests

def delete_task(id: str):
    try:
        url = "https://api.recoupable.com/api/tasks"
        payload = {"id": id}
        headers = {"Content-Type": "application/json"}
        response = requests.delete(url, json=payload, headers=headers)
        response.raise_for_status()
        return response.json()
    except requests.exceptions.RequestException as error:
        print(f"Error deleting task: {error}")
        raise
```

```javascript [JavaScript]
async function deleteTask(id) {
  try {
    const response = await fetch("https://api.recoupable.com/api/tasks", {
      method: "DELETE",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ id }),
    });
    if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
    return await response.json();
  } catch (error) {
    console.error("Error deleting task:", error);
    throw error;
  }
}
```

```typescript [TypeScript]
interface DeleteTaskResponse {
  status: "success" | "error";
  error?: string;
}

async function deleteTask(id: string): Promise<DeleteTaskResponse> {
  const response = await fetch("https://api.recoupable.com/api/tasks", {
    method: "DELETE",

    body: JSON.stringify({ id }),
  });
  if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
  return (await response.json()) as DeleteTaskResponse;
}
```
