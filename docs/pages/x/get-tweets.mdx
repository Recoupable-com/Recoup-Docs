# Twitter Get User Tweets API

Retrieve tweets from a specific Twitter user using the `getTweets` method from [agent-twitter-client](https://github.com/elizaOS/agent-twitter-client) (no Twitter API key required).

## Endpoint

```http
GET https://api.recoupable.com/api/x/tweets
```

## Parameters

| Name      | Type   | Required | Description                                                      |
| --------- | ------ | -------- | ---------------------------------------------------------------- |
| user      | string | Yes      | The Twitter username or user ID whose tweets should be returned. |
| maxTweets | int    | No       | The maximum number of tweets to return. Defaults to `200`.       |

## Request Examples

:::code-group

```bash [cURL]
curl -X GET "https://api.recoupable.com/api/x/get-tweets?user=elonmusk&maxTweets=50" \
  -H "Content-Type: application/json"
```

```python [Python]
import requests

def get_user_tweets(user, maxTweets=200):
    url = "https://api.recoupable.com/api/x/get-tweets"
    params = {"user": user}
    if maxTweets is not None:
        params["maxTweets"] = maxTweets
    response = requests.get(url, params=params)
    response.raise_for_status()
    return response.json()

# Example usage:
# get_user_tweets("elonmusk", maxTweets=50)
```

```javascript [JavaScript]
async function getUserTweets(user, maxTweets = 200) {
  const params = new URLSearchParams({ user });
  if (maxTweets !== undefined) params.append("maxTweets", maxTweets.toString());
  const response = await fetch(
    `https://api.recoupable.com/api/x/get-tweets?${params}`,
    { headers: { "Content-Type": "application/json" } }
  );
  if (!response.ok) throw new Error("Twitter getTweets failed");
  return await response.json();
}
```

## Workflow Guide

This endpoint is typically used as part of a workflow to:

1. Retrieve tweets from a specific user by username or user ID
2. Display tweet results with author and engagement info
3. Optionally, use tweet IDs to fetch more details or replies

## Notes

- This endpoint uses the `getTweets` method from [agent-twitter-client](https://github.com/elizaOS/agent-twitter-client) and does **not** require a Twitter API key.
- The `maxTweets` parameter defaults to 200 if not provided.
- Rate limits may apply (see [agent-twitter-client docs](https://github.com/elizaOS/agent-twitter-client)).
- Results are real-time but may be subject to unofficial API limitations.
- For advanced options, see the [agent-twitter-client documentation](https://github.com/elizaOS/agent-twitter-client).
